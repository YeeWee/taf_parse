# TAF 格式说明

## 概述

TAF (Terminal Aerodrome Forecast) 是机场天气预报的标准代码格式，由世界气象组织 (WMO) 和国际民航组织 (ICAO) 定义。

## TAF 报文结构

一个完整的 TAF 报文包含以下部分：

```
TAF <ICAO> <发布时间> <有效期> <初始天气> [变化组...]
```

## 各部分详解

### 1. 报文头
- `TAF` - 固定标识，表示这是一个 TAF 报文

### 2. 机场 ICAO 代码
- 4个字母的机场代码
- 例如：ZBAA（北京首都）、ZSPD（上海浦东）

### 3. 发布时间
- 格式：`DDHHMMZ`
- DD - 日期（UTC）
- HH - 小时（UTC）
- MM - 分钟（UTC）
- Z - 表示 UTC 时间
- 例如：`051100Z` 表示 5日 11:00 UTC

### 4. 有效期
- 格式：`DDHH/DDHH`
- 开始日时 / 结束日时
- 例如：`0512/0618` 表示从 5日12时 到 6日18时

### 5. 初始天气状态
初始天气状态包括以下要素，顺序通常为：

#### 5.1 风
- 格式：`dddffGffMPS` 或 `dddffGffKT`
- ddd - 风向（度），`VRB` 表示可变
- ff - 风速
- Gff - 阵风（可选）
- MPS/KT - 单位（米/秒 或 节）
- 例如：`18004MPS` 风向180度，风速4米/秒
- 例如：`32010G18MPS` 风向320度，风速10，阵风18

#### 5.2 能见度
- 单位：米
- `9999` 表示能见度大于10公里
- 例如：`6000` 能见度6000米
- 例如：`1500` 能见度1500米

#### 5.3 CAVOK
- `CAVOK` = Ceiling And Visibility OK
- 表示：能见度≥10公里，无重要天气，云高≥1500英尺或无云

#### 5.4 天气现象
由 2-9 个字符组成，可多个组合：

**强度/描述前缀：**
- `-` 小
- `+` 强
- `VC` 附近
- `RE` 近来（已停止）

**描述词：**
- `MI` 浅
- `PR` 部分
- `BC` 补丁
- `DR` 低吹
- `BL` 高吹
- `SH` 阵雨
- `TS` 雷暴
- `FZ` 冻

**天气现象：**
- `DZ` 毛毛雨
- `RA` 雨
- `SN` 雪
- `SG` 米雪
- `IC` 冰晶
- `PL` 冰粒
- `GR` 冰雹
- `GS` 小冰雹/雪粒
- `UP` 未知降水

**能见度障碍：**
- `BR` 轻雾
- `FG` 雾
- `FU` 烟
- `VA` 火山灰
- `DU` 浮尘
- `SA` 沙
- `HZ` 霾

**其他现象：**
- `PO` 尘/沙旋风
- `SQ` 飑
- `FC` 漏斗云
- `SS` 沙暴
- `DS` 尘暴

**示例：**
- `SHRA` 阵雨
- `TSRA` 雷暴伴雨
- `-SN` 小雪
- `+TSRA` 强雷暴伴雨
- `FZRA` 冻雨
- `BR` 轻雾

#### 5.5 云
- 格式：`aaannn[tt]`
- `aaa` - 云量
  - `SKC` 晴空
  - `FEW` 少云（1/8 - 2/8）
  - `SCT` 疏云（3/8 - 4/8）
  - `BKN` 多云（5/8 - 7/8）
  - `OVC` 阴天（8/8）
- `nnn` - 云高（百英尺）
- `tt` - 云类型（可选）
  - `CB` 积雨云
  - `TCU` 浓积云

**示例：**
- `SCT030` 疏云，云高3000英尺
- `BKN010` 多云，云高1000英尺
- `OVC040CB` 阴天，云高4000英尺，积雨云

### 6. 变化组

变化组描述天气在预报期内的变化。

#### 6.1 FM - From（从何时起）
- 格式：`FMDDHHMM`
- 表示从指定时间起天气变为新状态
- 例如：`FM051400` 从5日14:00起

#### 6.2 BECMG - Becoming（逐渐变为）
- 格式：`BECMG DDHH/DDHH`
- 表示在指定时间段内天气逐渐变化
- 例如：`BECMG 0514/0516` 在5日14-16时之间逐渐变化

#### 6.3 TEMPO - Temporary（暂时）
- 格式：`TEMPO DDHH/DDHH`
- 表示天气暂时变化，每次不超过1小时，总共不超过一半时间
- 例如：`TEMPO 0516/0520` 5日16-20时之间有时出现

#### 6.4 PROB - Probability（概率）
- 格式：`PROBnn TEMPO DDHH/DDHH`
- nn - 概率百分比（30或40）
- 表示有nn%的概率出现暂时变化
- 例如：`PROB30 TEMPO 0518/0522` 30%概率在5日18-22时出现

#### 6.5 INTER - Intermittent（间歇）
- 某些地区使用，表示间歇性变化

### 7. 温度预报（可选）
- 格式：`TXt/TTtZ TNt/TTtZ`
- `TX` - 最高温度
- `TN` - 最低温度
- `t` - 日期
- `TT` - 温度（摄氏度）
- 例如：`TX06/28Z TN05/18Z` 最高28度（6日），最低18度（5日）

## 完整示例

### 示例1：北京首都机场
```
TAF ZBAA 051100Z 0512/0618 18004MPS 6000 SCT030
BECMG 0514/0516 32010G18MPS 3000 SHRA BKN010
TEMPO 0516/0520 1500 TSRA
BECMG 0522/0600 20005MPS 9999 NSW SCT040
```

**解析：**
- 机场：ZBAA 北京首都
- 发布时间：5日11:00 UTC
- 有效期：5日12时 - 6日18时
- 初始天气（12-14时）：风向180度，风速4m/s，能见度6000米，疏云3000英尺
- 14-16时逐渐变为：风向320度，风速10m/s阵风18，能见度3000米，阵雨，多云1000英尺
- 16-20时有时出现：能见度1500米，雷暴伴雨
- 22时-次日00时逐渐变为：风向200度，风速5m/s，能见度>10公里，无重要天气，疏云4000英尺

### 示例2：上海浦东机场
```
TAF ZSPD 050600Z 0509/0615 09003MPS CAVOK
BECMG 0512/0514 15006MPS
TEMPO 0515/0519 3000 BR
BECMG 0521/0523 08002MPS
```

**解析：**
- 机场：ZSPD 上海浦东
- 发布时间：5日06:00 UTC
- 有效期：5日09时 - 6日15时
- 初始天气：风向090度，风速3m/s，CAVOK（良好）
- 12-14时逐渐变为：风向150度，风速6m/s
- 15-19时有时出现：能见度3000米，轻雾
- 21-23时逐渐变为：风向080度，风速2m/s

## 注意事项

1. **时间均为 UTC** - TAF 中所有时间都是协调世界时
2. **单位转换** - 注意 m/s 与 kt，米与英尺/ statute mile 的转换
3. **CAVOK 优先级** - CAVOK 出现时，不需要能见度、天气、云组
4. **NSW** - No Significant Weather，无重要天气
5. **NCD** - No Cloud Detected，无云
